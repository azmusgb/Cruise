How to Crop an SVG Image

Posted 5 Years Ago

A fast, editor-only workflow for trimming whitespace and focusing on the exact SVG artwork you want.

Cropping SVGs with viewBox

A quick guide to framing your vector art without extra tools.

Quite often, web developers have a great SVG image that they want to use but would prefer a cropped version of it. It could be that there's too much whitespace or they only want to use a specific part of the graphic. Take this vector illustration, for instance.

Suppose you wish to hide the smoke clouds and also want to remove the whitespace to show the rocket dead-center. There are a bunch of tools you can use to crop the image. You can use Inkscape, or some online utility. However, the fastest way is to edit the viewBox of SVG in a text editor. It is far more efficient than importing the SVG in software and then exporting it again.

ViewBox in a Nutshell

The viewBox is easiest to understand as a camera on an infinite canvas. With viewBox="x y width height", you control which part of the canvas is visible and how it scales.

For instance, the rocket illustration above has a viewport of 0 168.4 940.7 724. If you double the width and height, the canvas stretches and you get empty space around the artwork.

Cropping SVG Using viewBox

Editing the viewBox attribute is an easy way to crop an SVG image. Open the SVG file in a text editor and start tweaking the numbers. Finding the right value is a bit of guesswork, but you get there in 3–4 tries.

Note: It's important that you don't use the height attribute while adjusting these numbers as the browser tries to scale both the width and height of the SVG.

Start with the x attribute. The default viewport is 0 168.4 940.7 724. We edit x to be close to the rocket:

Try x = 400 viewBox="400 168.4 940.7 724"

Too much.

Try x = 300 viewBox="300 168.4 940.7 724"

Almost right.

Next, adjust y to sit just above the top of the rocket:

Try y = 300 viewBox="300 300 940.7 724"

Still too much.

Try y = 280 viewBox="300 280 940.7 724"

Closer.

Repeat for width and height. Reduce width to reach the right edge of the rocket, and height to trim the bottom space:

Width around 380 viewBox="300 280 380 724"
Height around 500 viewBox="300 280 380 500"

Final cropped viewBox: 300 280 380 500.

ViewBox Approach vs Others

Some may feel that guesswork isn't the best approach to do a trivial image manipulation operation. That is fair, but this workflow is still the fastest: adjust numbers, refresh, repeat.

Vector design software aren't always perfect in interpreting SVGs and exporting them properly. Using a text editor is faster and saves you more trouble than it gives. Give it a try!
